## 実装メモ

### WCP OctahedralMap border copy

`wcp_fill_probe_octmap_atlas_border_cs.hlsl` の境界コピーは、
**OctahedralMap を単純なバイリニアサンプリングで扱えるようにするために、境界 texel に反対側の対応 texel をコピーする**処理。

- 各 probe の OctahedralMap は atlas 上で **1 texel の border** を持つ
- 内側の有効領域をそのまま `Sample` / `SampleLevel` したいので、
  border には **対向辺 / 対向角**の値を複製して seam を埋める
- こうしておくと、境界近傍でも atlas 内だけで bilinear 補間が閉じる

#### 最小構成のサンプル

以下は、**4 つの角 + 4 つの辺**だけを最小限で処理するサンプル。
`W` は border 込みの 1 probe 幅 (`k_wcp_probe_octmap_width_with_border`) を想定する。

```cpp
// probe ごとの atlas 左上
const int2 base = octmap_atlas_texel_pos_min;
const int W = k_wcp_probe_octmap_width_with_border;

// 4 corners
dst[base + int2(0,     0    )] = dst[base + int2(W - 2, W - 2)];
dst[base + int2(W - 1, 0    )] = dst[base + int2(1,     W - 2)];
dst[base + int2(0,     W - 1)] = dst[base + int2(W - 2, 1    )];
dst[base + int2(W - 1, W - 1)] = dst[base + int2(1,     1    )];

// 4 edges
for (int i = 1; i < W - 1; ++i)
{
    dst[base + int2(i,     0    )] = dst[base + int2(W - 1 - i, 1        )]; // top
    dst[base + int2(0,     i    )] = dst[base + int2(1,         W - 1 - i)]; // left
    dst[base + int2(W - 1, i    )] = dst[base + int2(W - 2,     W - 1 - i)]; // right
    dst[base + int2(i,     W - 1)] = dst[base + int2(W - 1 - i, W - 2    )]; // bottom
}
```

#### 対応の考え方

- **corner** は対角の内側 texel をコピーする
- **edge** は反対側の 1 texel 内側を、**左右 / 上下反転つき**でコピーする
- OctahedralMap の折り返しでは軸方向が反転するため、単純な同 index コピーではなく
  `W - 1 - i` で向きを合わせるのがポイント

#### 目的

この処理を入れることで、

1. shader 側で border 特例を書かずに済む
2. atlas 内の 1 probe をそのまま bilinear sample しやすい
3. OctahedralMap seam の破綻を軽減できる

#### 注意

- この border copy は **probe 単位で閉じた複製**であり、隣接 probe から値を持ってくる処理ではない
- atlas packing を変えた場合は、
  - border 幅
  - 内側有効領域の始点
  - 反転方向
  を再確認すること
